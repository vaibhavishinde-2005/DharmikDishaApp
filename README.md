# 🛕 Dharmik Disha

Dharmik Disha is an Android application developed using Kotlin and Firebase that allows users to book pandits for poojas and religious services easily.


- User Login & Registration
- Pandit Login System
- Browse Pandits
- Book Pandits for Pooja
- Real-time Booking Status
- Accept / Reject Booking
- Payment Integration
- Booking History
- Firebase Realtime Database
- Modern Android UI

## 🛠️ Tech Stack

- Kotlin
- Firebase Authentication
- Firebase Realtime Database
- Android Studio
- RecyclerView
- Material Design
